# 关于此中文翻译版

你好，感谢你访问 Rust 语言速查表的中文版！

## 项目缘起

[cheats.rs](https://cheats.rs) 是由 [Ralf Biedert](https://github.com/ralfbiedert) 创建的一个非常出色的 Rust 学习资源。它以其极高的信息密度和全面的内容覆盖，成为了许多开发者快速查找和学习 Rust 语法的首选工具。

为了消除语言障碍，让更多热爱 Rust 的中文开发者能够方便地使用这个工具，我们启动了这个社区翻译项目。

## 翻译是如何完成的？

本项目的大部分初始翻译工作由 Google 的大型语言模型 **Gemini** 在 [jhq223](https://github.com/jhq223) 的指导下完成。随后，由人工进行细致的校对、术语统一和格式调整，以确保翻译的准确性和可读性。

我们深知，技术文档的翻译需要严谨和精确。尽管我们已经尽力，但其中仍可能存在疏漏或可以改进之处。
